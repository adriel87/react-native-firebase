# Pasos aseguir durante la instalacion

1. Se supone que ya conoces sobre react native y como crear un proyecto en el mismo

- si tienes dudas mira la [documentacion oficial](https://reactnative.dev/docs/environment-setup)

2. crea un nuevo proyecto en [firebase](https://firebase.google.com/?hl=es)

3. instala las siguientes dependencias en tu proyecto

```Shell
~$ npm install --save @react-native-firebase/app
```

- esta libreria es para manejar de forma mas sencilla firebase dentro de proyectos de react native
